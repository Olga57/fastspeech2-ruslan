# 🎙️ FastSpeech 2 — Russian Text-to-Speech

Данный репозиторий содержит реализацию нейросетевой модели синтеза речи **FastSpeech 2** для русского языка. Проект выполнен в рамках домашнего задания по курсу глубокого обучения (Deep Learning).

Модель переводит входной текст (графемы/фонемы) в мел-спектрограммы, которые затем преобразуются в аудиосигнал с помощью предобученного вокодера HiFi-GAN.

---

## 🌟 Выполненные требования ТЗ (Чек-лист для проверки)

В проекте строго соблюдены все требования задания:
* ✅ **Архитектура написана с нуля:** Реализованы кастомные модули `Multi-Head Attention`, `Positionwise Feed-Forward`, `FFT-Block` и `Variance Adaptor`. Готовый модуль `torch.nn.Transformer` **не использовался**.
* ✅ **Интеграция MFA (Бонусные баллы):** Для выравнивания текста и аудио (alignments), а также получения таргетов длительности (duration) использовался Montreal Forced Aligner (MFA), что значительно повысило качество по сравнению со стандартными attention-матрицами.
* ✅ **Variance Adaptor:** Модель успешно предсказывает `Pitch` (высоту тона), `Energy` (энергию/громкость) и `Duration` (длительность фонем).
* ✅ **Логирование в Weights & Biases (W&B):** Во время обучения логировались все компоненты функции потерь (`total_loss`, `mel_loss`, `pitch_loss`, `dur_loss`), а также валидационные спектрограммы и синтезированные аудиофайлы (MOS-тест).
* ✅ **Вокодер:** Использован предобученный HiFi-GAN (через `speechbrain`).
* ✅ **Git LFS:** Тяжелые веса модели (`.pth`) хранятся прямо в репозитории с помощью Git Large File Storage и подтягиваются автоматически.

## 🚀 Запуск демонстрационного ноутбука (Инференс)

Для удобного тестирования синтеза подготовлен полностью автоматизированный ноутбук `demo.ipynb`. Он адаптирован для запуска в Google Colab и не требует ручного редактирования кода.

**Ключевые особенности демо:**
* **Автоматизация:** Скрипт самостоятельно загружает чекпоинты (через Git LFS), вокодер HiFi-GAN и настраивает окружение.
* **Интерактивный UI:** Генерация управляется через графический интерфейс (виджеты).
* **Variance Adaptor в действии:** Добавлены ползунки для управления характеристиками речи (Speed, Pitch, Energy) в реальном времени.
* **Фонетический маппинг:** Встроенный транскрибатор IPA (Международный фонетический алфавит) для точного произношения тестовых фраз.

**Инструкция по проверке:**
1. Откройте файл `demo.ipynb` в Google Colab.
2. Обязательно включите аппаратный ускоритель: **Среда выполнения -> Сменить среду выполнения -> T4 GPU** (Runtime -> Change runtime type -> T4 GPU).
3. В верхнем меню выберите **Среда выполнения -> Выполнить всё** (Run all).
4. *Примечание:* На первом шаге ноутбук установит зависимости и выполнит мягкий перезапуск среды выполнения — это штатная процедура.
5. Прокрутите ноутбук в самый низ ко второй ячейке: выберите фразу из выпадающего списка (или введите свою), настройте ползунки характеристик звука и нажмите кнопку **«Синтезировать 🎙️»**.
